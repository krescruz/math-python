Matemáticas con python

Variables Numéricas enteras y reales
---
Python tiene dos tipos de variables numéricas: Enteras, aquellas sin decimales int(integer) y las Reales, con decimales y de tipo float. 
```
>>>entero=5
>>>real=5.0
```

Operaciones básicas
---
Suma
```
>>>2+2
4
```
```
>>>2+2.2
4.2
```
Resta
```
>>>2-2
0
```
Multiplicación
```
>>>5*3
15
```
División
  Números enteros
```
>>>6/4
1
```
  Números decimales
```
>>>6/4.0
1.5
```
```
>>>6/float(4)
1.5
```
